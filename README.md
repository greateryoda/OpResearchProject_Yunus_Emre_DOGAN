Etudiant: Yunus Emre DOGAN

## Structure du projet
    Examples/ : la repertoire contenant des divers Graphs comme example pour tester
    Main.java : programme principale a executer
    bin/ : la repertoire contenant fichiers .class
    out/ : la repertoire contenant les images des graphs de sortie
    src/ : la repertoire contenant les fichiers du projet

## Compilation et execution

### Compilation
    javac -d bin src/*.java

### Execution
    La syntaxe generale: java -cp bin Main Examples/<nom_fichier>
    Example: java -cp bin Main Examples/graph_data.txt
